# 安全性测试工具

一个用于测试密码和加密方式安全性的Python工具，支持密码爆破测试和加密方式测试。

## 版本信息

- **当前版本**: v0.1.2
- **发布日期**: 2026-02-13
- **更新内容**: 添加多种加密算法支持，包括SHA-3、BLAKE2、MD4、RIPEMD160等30+种哈希算法

## 功能特性

### 密码爆破测试
- **字典攻击**：使用内置字典或自定义字典进行密码测试
- **暴力破解**：基于字符集的全排列组合测试
- **掩码攻击**：支持用户定义密码模式
- **性能统计**：记录破解时间和尝试次数

### 加密方式测试
- **单加密方式爆破**：支持30+种加密算法，包括MD5、SHA1、SHA256、SHA512、SHA-3、BLAKE2等
- **混合加密爆破**：支持多层加密组合（如 MD5(SHA256(password))）
- **加密强度评估**：分析加密结果的安全性

### 支持的加密算法
- **MD系列**：MD4、MD5
- **SHA系列**：SHA1、SHA224、SHA256、SHA384、SHA512
- **SHA-3系列**：SHA3_224、SHA3_256、SHA3_384、SHA3_512
- **BLAKE系列**：BLAKE2b、BLAKE2s
- **RIPEMD系列**：RIPEMD160
- **其他算法**：Whirlpool、Tiger192_3、Snefru、GOST
- **校验和算法**：Adler32、CRC32、CRC32b
- **HAVAL系列**：HAVAL128_3、HAVAL160_3、HAVAL192_3、HAVAL224_3、HAVAL256_3

### 用户界面
- **图形交互**：提供友好的GUI界面
- **结果输出**：清晰展示测试结果和安全性评估

## 系统要求

- Python 3.14+
- tkinter（Python标准库，用于GUI）

## 安装与运行

1. **克隆或下载项目**
   ```bash
   git clone <repository-url>
   cd password-security-tester
   ```

2. **运行程序**
   ```bash
   python main.py
   ```

## 使用指南

### 密码测试
1. 在"密码测试"选项卡中输入目标哈希值
2. 选择攻击方式（字典攻击、暴力破解或掩码攻击）
3. 根据攻击方式设置相应参数：
   - 字典攻击：选择字典文件路径
   - 暴力破解：设置字符集、最小长度和最大长度
   - 掩码攻击：设置掩码和字符集
4. 选择哈希算法
5. 点击"开始测试"按钮

### 加密测试
1. 在"加密测试"选项卡中输入目标哈希值
2. 选择加密方式（单加密或混合加密）
3. 根据加密方式设置相应参数：
   - 单加密：选择加密算法
   - 混合加密：设置加密链（如 md5>sha256）
4. 选择字典文件路径
5. 点击"开始测试"按钮

### 查看结果
- 在"测试结果"选项卡中查看测试过程和结果
- 结果包括：破解状态、密码（如果成功）、尝试次数和耗时

## 性能优化

- **多进程并行计算**：使用Python的multiprocessing模块充分利用多核CPU，提高破解速度
- **配置管理**：支持自定义性能参数，如多进程数量、内存限制、缓存大小等
- **批处理机制**：采用分批次处理密码，减少进程间通信开销
- **内存使用优化**：采用生成器和迭代器减少内存占用
- **计算资源合理分配**：根据系统资源调整进程数和线程数
- **缓存机制**：添加哈希结果缓存，避免重复计算

## 扩展与定制

### 添加自定义字典
1. 创建一个文本文件，每行一个密码
2. 在测试时选择该文件作为字典文件路径

### 添加新的哈希算法
1. 修改 `utils/hashing.py` 文件中的 `calculate_hash` 方法
2. 在 `main.py` 中更新算法选择列表

## 注意事项

- 暴力破解和掩码攻击可能会消耗大量计算资源，请谨慎使用
- 对于强密码和复杂加密方式，破解可能需要很长时间甚至无法完成
- 本工具仅用于教育和合法测试目的，请勿滥用
